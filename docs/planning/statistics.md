# 통계 도표

## 배경
일별/주별 수유 패턴을 파악해 건강한 양육 리듬을 만들 수 있어야 한다.

## 요구사항
- [ ] 일별 수유 횟수 막대 차트 (최근 7일)
- [ ] 주별 총 수유량 라인 차트 (최근 4주)
- [ ] 평균 수유 간격 숫자 표시
- [ ] 차트 탭 → 해당 날짜 기록 목록으로 이동

## 기술 설계
- `useFeedingStats` 훅이 집계 로직을 담당
- `victory-native` 로 렌더
- 집계는 순수 함수(`aggregateByDay`, `aggregateByWeek`)로 분리 → 단위 테스트

## 테스트 계획
- [ ] aggregateByDay: 기록 5개 → 일별 카운트 배열
- [ ] aggregateByWeek: 주 경계 (일요일→월요일) 처리
- [ ] 평균 수유 간격: 기록 0개 → null 반환

## 마일스톤
- [ ] M1: 일별 막대 차트
- [ ] M2: 주별 라인 차트
- [ ] M3: 평균·최대 지표
